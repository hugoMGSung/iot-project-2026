# iot-project-2026
IoT 개발자 프로젝트(미니, 파이널) 저장소

## 팀프로젝트 

### 팀조직 생성

- 프로필 > Organizations > New Organization 클릭
- Free 플랜 선택
    - Organization name : 팀명(영어)
    - Contact email : 로그인 이메일
    - My personal account 체크 
    - Team of Service 체크
    - Next 버튼 클릭

- Add organization members
    - Search by username, full name or email address : 팀원들 이메일주소 추가
    - 팀원 이메일로 인증 메일 도착, 인증
    - Complete...

#### 팀 권한 부여

- Profile > Setting > Collaborators and teams > Add Team
    - Read/Write, Maintain, Admin, Triage(Issue만 관리)
- Member privileges > Base Permissions Write로 지정

#### 팀리포지토리 생성

- 일반 리포지토리 생성과 동일
- 브랜치 리포지토리 생성 머지 주의
- Pull/Push로 생기는 충돌(Conflict) 주의 요함 - 항상 Pull 다음에 Push 또는 `동기화`!

#### 깃허브 조직

- 팀리포지토리 내 기능 활용
    - 프로젝트 
    - 이슈
    - 위키 

### 프로젝트 진행

#### 순서

- 주제 선정(1) -> 역할 분담 -> 요구사항 정의(2) -> 장비 선정/구매(3) -> UI 설계/DB 설계(4) -> 개발환경 설정 -> 기능 개발(디버그) -> 코드테스트 -> 코드 리뷰 -> 최종 발표

#### 2 요구사항 정의

- 팀리포지토리 활용
- 노션 활용

#### 3 장비 선정/구매

- 엑셀로 정리

#### 4. UI설계/DB설계
